## The Function Machine

Imagine a machine on your kitchen table. It has a slot on top where you drop in a number. Inside, the machine does something to that number. Then out pops a new number.

```
         INPUT
           |
           v
      +----------+
      |  THE     |
      | MACHINE  |
      +----------+
           |
           v
         OUTPUT
```

That's a **function**. You put a number in, you get a number out.

The rule the machine follows is always the same. If you put in 3, you always get the same answer. If you put in 5, you always get the same answer. The function is consistent.

---

## Simple Functions

Let's build some function machines.

**Function 1: "Double It"**

Whatever number you put in, the machine doubles it.

- Put in 2, get out 4
- Put in 5, get out 10
- Put in 10, get out 20
- Put in -3, get out -6

We can write this as a rule: **f(x) = 2x**

Translation: "The function of x equals two times x."

Don't panic about the notation. All it means is: take whatever number you have (we call it x), and multiply it by 2. That's the output.

**Function 2: "Add Three"**

Whatever number you put in, add 3 to it.

- Put in 2, get out 5
- Put in 0, get out 3
- Put in -1, get out 2
- Put in 10, get out 13

The rule: **f(x) = x + 3**

Translation: "Take your number, add 3, that's your answer."

**Function 3: "Square It"**

Multiply the number by itself.

- Put in 2, get out 4 (because 2 × 2 = 4)
- Put in 3, get out 9 (because 3 × 3 = 9)
- Put in 5, get out 25
- Put in -2, get out 4 (because -2 × -2 = 4)

The rule: **f(x) = x × x** (or we can write **f(x) = x²**)

---

## Why "f(x)"?

The letter **f** just stands for "function." The letter **x** is the input number.

So **f(x) = 2x** means: "This function takes an input (x) and gives you back double that input."

When we write **f(3)**, we mean: "What does the function give you when you put in 3?"

If **f(x) = 2x**, then:
- **f(3) = 6** (double of 3)
- **f(10) = 20** (double of 10)
- **f(0) = 0** (double of 0)

Think of it like a recipe. The recipe is **f(x) = 2x**. The ingredient is whatever number you choose for x. The result is what comes out.

## Making a Table

For any function, you can make a table showing inputs and outputs.

Let's try **f(x) = x + 5** (add 5 to whatever you put in):

```
 Input (x) | Output f(x)
-----------+------------
     0     |      5
     1     |      6
     2     |      7
     3     |      8
     4     |      9
```

See the pattern? Each time x goes up by 1, the output goes up by 1. That's because we're just adding 5 to x.

## Graphing Functions

Here's where it gets beautiful. Every function can be drawn as a graph.

Remember from F-06: a graph plots points (x, y). For a function, **x** is the input and **y** is the output.

So **f(x) = y**. The function tells you what y is for any x you choose.

**Example:** Graph **f(x) = 2x**

Make a table first:

```
 x  |  f(x)
----+------
-2  |  -4
-1  |  -2
 0  |   0
 1  |   2
 2  |   4
 3  |   6
```

Now plot these points: (-2, -4), (-1, -2), (0, 0), (1, 2), (2, 4), (3, 6)

```
    6 |                       *
      |
    4 |                   *
      |
    2 |               *
      |
    0 +-------*-------+-------
      |   *           |
   -2 | *             |
      |               |
   -4 *               |
      |               |
    --+---------------+-------
     -2  -1   0   1   2   3
```

Connect the dots and you get a straight line. This function creates a line because it grows steadily.

---

## Straight Lines and Curves

Some functions make straight lines. Some make curves.

**Straight line functions** grow (or shrink) by the same amount every step:
- **f(x) = x** (a diagonal line)
- **f(x) = 2x** (a steeper diagonal)
- **f(x) = x + 3** (diagonal line, shifted up)

**Curved functions** grow faster or slower as x changes:
- **f(x) = x²** (gets steeper as you go right)
- **f(x) = 1/x** (gets flatter as you go right)

Here's **f(x) = x²** graphed:

```
    9 |              *
      |
    4 |          *       *
      |
    1 |       *             *
      |
    0 +-------*--------------
      |
      +--+--+--+--+--+--+--+--
      -3 -2 -1  0  1  2  3
```

Notice how it curves upward? At x = 0, the output is 0. But at x = 3, the output is 9. It accelerates.

This is the shape of **many things in nature**: falling objects, swinging pendulums, bouncing springs. Curved functions describe accelerating motion.

---

## Why Functions Matter in Physics

Physics is the study of how things change. And functions describe change mathematically.

- **How far has a ball fallen?** There's a function for that (involving time).
- **How fast is a car going?** There's a function for that (speed over time).
- **How does a pendulum swing?** There's a function for that (position over time).

And waves — the heartbeat of physics — are functions too. A wave is just a special function that goes up and down and up and down forever. You're almost there.

---

## Building Your Intuition

When you see **f(x) = 2x**:
- Think: "For every 1 step right, go 2 steps up."
- That's the slope of the line.

When you see **f(x) = x + 5**:
- Think: "Whatever x is, the output is 5 higher."
- That shifts the whole graph up by 5.

When you see **f(x) = x²**:
- Think: "Squaring makes things grow fast."
- That creates a curve, not a line.

You're building a mental library of function shapes. Soon you'll see a function and instantly know what its graph looks like.
