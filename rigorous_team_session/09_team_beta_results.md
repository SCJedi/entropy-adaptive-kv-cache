## 1. Method and Setup

### 1.1 Formalism

We work exclusively in the Weyl algebra formulation on Minkowski spacetime. The algebra W(S, sigma) is generated by unitaries W(f) for f in the symplectic space S of real Klein-Gordon solutions, satisfying:

```
W(f)W(g) = exp(-i sigma(f,g)/2) W(f+g)
W(f)* = W(-f)
```

The symplectic form is:
```
sigma(f, g) = integral_Sigma (f nabla_n g - g nabla_n f) dSigma
```

The mode vacuum state omega_0 on W(S, sigma) is the unique Poincare-invariant quasi-free Hadamard state:
```
omega_0(W(g)) = exp(-||g||^2/4)
```
where ||g||^2 is the one-particle norm.

### 1.2 The Cell Vacuum State

The cell vacuum omega_Omega is defined as a coherent displacement of omega_0:
```
omega_Omega(W(g)) = omega_0(W(g)) * exp(i sigma(f_total, g))
```

where f_total is a Klein-Gordon solution representing the total classical displacement field.

### 1.3 The Two-Point Function

For the cell vacuum:
```
W_Omega(x, y) = omega_Omega(phi(x) phi(y)) = W_0(x, y) + F(x)F(y)
```

where:
- W_0(x,y) = omega_0(phi(x) phi(y)) is the Fock vacuum Wightman function
- F(x) = omega_Omega(phi(x)) is the one-point function (classical displacement field)

This decomposition is exact for coherent states (cross terms vanish because omega_0(phi(x)) = 0). This is Theorem 5.3 of the AQFT foundations document.

### 1.4 The Stress-Energy Operator

For a real massive scalar field with minimal coupling (xi = 0):
```
T_{mu nu} = nabla_mu phi nabla_nu phi - (1/2) g_{mu nu} (nabla^alpha phi nabla_alpha phi + m^2 c^2 phi^2 / hbar^2)
```

### 1.5 Hadamard Renormalization

```
<T_{mu nu}(x)>_ren = lim_{x' -> x} D_{mu nu'}[W_Omega(x,x') - H(x,x')] + geometric counterterms
```

where H(x,x') is the Hadamard parametrix (state-independent, determined by geometry and mass).

## 6. Wald Ambiguity Analysis (Step E)

### 6.1 The Wald Ambiguity on Flat Spacetime

On flat spacetime: G_{mu nu} = 0, I_{mu nu} = 0, J_{mu nu} = 0. Only alpha_1 g_{mu nu} survives.

### 6.2 Effect on w

The alpha_1 g_{mu nu} term has w = -1 by construction. Adding it to any state's stress-energy shifts both rho and p:
```

p -> p - alpha_1
```

This changes the numerical value of w but CANNOT make w = -1 for the total unless alpha_1 dominates completely.

p_state = -rho_state
w_state = -1

**Contradiction.** Therefore: if the state-dependent part has w != -1, no choice of alpha_1 can make the total w = -1.
