# F-02: True and False

Welcome! You're about to learn one of the most powerful ideas in all of thinking: the difference between true and false. By the end of this lesson, you'll know how to check if a statement is true and why this matters. You can do this.

## What Does "True" Mean?

A statement is TRUE if it matches reality. If it describes the world as it actually is.

Look at these statements:

**TRUE statements:**
- The sky is blue (on a clear day)
- 2 + 2 = 4
- Apples are fruit
- Ice is cold
- Dogs have four legs

Why are these true? Because when you check them against reality, they match. Count 2 + 2 on your fingers. It's 4. Look at the sky on a clear day. It's blue.

**FALSE statements:**
- The sky is green (usually it's not)
- 2 + 2 = 5
- Apples are vegetables
- Ice is hot
- Dogs have six legs

Why are these false? Because they DON'T match reality. When you check, they're wrong.

## How to Check If Something Is True

There are different ways to check if a statement is true:

**Observation (Look and See):**

```
Statement: "There are three apples on the table"
How to check: Count the apples
Result: If you count three, it's true. If not, it's false.
```

**Logic (Think It Through):**

```
Statement: "If all dogs are animals, and Spot is a dog, then Spot is an animal"
How to check: Follow the logic
Result: This must be true if the starting facts are true
```

**Calculation (Do the Math):**

```
Statement: "7 × 8 = 56"
How to check: Multiply 7 × 8
Result: 7 × 8 actually equals 56, so it's true
```

**Testing (Try It Out):**

```
Statement: "Sugar dissolves in water"
How to check: Put sugar in water and stir
Result: The sugar disappears (dissolves), so it's true
```

## Statements That Could Be Either

Some statements are sometimes true and sometimes false:

**"It is raining"**
- True right now if rain is falling
- False right now if rain is not falling
- Could be true tomorrow, or false tomorrow

**"Sarah is wearing a red shirt"**
- True today if she chose a red shirt
- False tomorrow if she wears blue

**"The temperature is above 70 degrees"**
- Depends on where you are and when you measure

These statements can be true or false depending on the situation. We need more information to check them.

## Statements We Don't Know Yet

Some statements HAVE an answer (true or false), but we don't know which yet:

**"It will rain tomorrow"**
- This is either true or false
- We don't know which until tomorrow comes
- But it's not "maybe" - it will either rain or it won't

**"There is life on planets outside our solar system"**
- This is either true (life exists out there) or false (it doesn't)
- We haven't discovered it yet, but the answer exists

**"The 1000th digit of pi is 7"**
- Pi has a 1000th digit
- That digit is either 7 or it isn't
- We can calculate it if we want to know

These are different from "It is raining." For rain, the truth changes. For these, the truth is fixed - we just don't know it yet.

## Why True and False Matter

Knowing what's true and false helps you:

**Make good decisions:**
```
TRUE: "This milk expired last week"
ACTION: Don't drink it

FALSE: "Jumping off a roof with a bedsheet parachute is safe"
ACTION: Don't try it
```

**Solve problems:**
```
Problem: What is 24 ÷ 6?
Try: 24 ÷ 6 = 4
Check: Is "6 × 4 = 24" true?
Yes! So our answer is correct.
```

**Learn from mistakes:**
```
You thought: "Fish can live on land"
You tested: Took a fish out of water
Result: Fish can't breathe - it needs water
Now you know: Your thought was FALSE
You learned: Fish need water to survive
```

## Opposite Statements

Every statement has an opposite:

```
Statement: "The door is open"
Opposite: "The door is NOT open" (same as "the door is closed")

Statement: "5 is bigger than 3"
Opposite: "5 is NOT bigger than 3" (which is false)

Statement: "All birds can fly"
Opposite: "NOT all birds can fly" (which is true - penguins can't fly!)
```

Here's an important rule:

**If a statement is TRUE, its opposite is FALSE.**
**If a statement is FALSE, its opposite is TRUE.**

They can't both be true. They can't both be false. It's always one or the other.

## Practice With True and False

Let's practice identifying true and false:

```
The sun is hot               → TRUE (you can feel it)
Cats are reptiles            → FALSE (cats are mammals)
10 is an even number         → TRUE (ends in 0)
Water freezes at 200°C       → FALSE (it freezes at 0°C)
Humans need oxygen           → TRUE (we can't survive without it)
1 + 1 = 3                    → FALSE (1 + 1 = 2)
```

Notice: We can EXPLAIN why each one is true or false. We don't just guess.

1. a) TRUE (9 + 6 really is 15)
