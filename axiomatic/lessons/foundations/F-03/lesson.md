# F-03: If-Then Statements

Welcome! You're about to learn how to predict the future using logic. No magic tricks, just clear thinking. By the end of this lesson, you'll understand if-then statements and how they help you figure out what happens next. You can do this.

## What Is an If-Then Statement?

An if-then statement connects two ideas: a condition (the IF part) and a consequence (the THEN part).

Here's a simple example:

```
IF it rains, THEN the ground gets wet.
```

This statement has two parts:
- **IF part (condition)**: "it rains"
- **THEN part (consequence)**: "the ground gets wet"

The statement tells you: WHEN the IF part happens, the THEN part happens too.

## If-Then in Everyday Life

You use if-then thinking all the time:

**Getting Dressed:**
```
IF it's cold outside, THEN I'll wear a jacket.
```

**Cooking:**
```
IF I put water in the freezer, THEN it will turn to ice.
```

**Studying:**
```
IF I practice piano every day, THEN I'll get better.
```

**Math:**
```
IF a number ends in 0, THEN it's divisible by 10.
```

Each time, the IF part tells you WHEN something is true, and the THEN part tells you what follows.

## Why If-Then Helps You Predict

If-then statements are powerful because they let you predict what will happen.

```
You know: IF you drop a glass, THEN it breaks.
You observe: You just dropped a glass.
You predict: The glass will break.
```

You didn't need to wait and see. The if-then rule told you what would happen.

Another example:

```
You know: IF a number is even, THEN it's divisible by 2.
You observe: 48 is even.
You conclude: 48 is divisible by 2.
```

Check it: 48 ÷ 2 = 24. The prediction was correct!

## The Parts of If-Then

Let's label the parts clearly:

```
IF [condition], THEN [consequence].
   └─ This is the HYPOTHESIS (the "suppose this is true" part)
                   └─ This is the CONCLUSION (what follows)
```

**Example 1:**
```
IF you touch fire, THEN you get burned.
   └─ Condition: touching fire
                    └─ Consequence: getting burned
```

**Example 2:**
```
IF a shape has three sides, THEN it's a triangle.
   └─ Condition: has three sides
                              └─ Consequence: it's a triangle
```

To use an if-then statement, you check if the IF part is true. If it is, then the THEN part must be true too.

## Chaining If-Then Statements

Here's where it gets really powerful. You can connect multiple if-then statements:

```
IF it rains, THEN the ground gets wet.
IF the ground gets wet, THEN the grass grows.

Therefore: IF it rains, THEN the grass grows.
```

We created a chain:

```
Rain → Wet ground → Growing grass
```

Let's try another:

```
IF you study, THEN you learn.
IF you learn, THEN you pass the test.

Therefore: IF you study, THEN you pass the test.
```

This is called a logical chain. Each THEN becomes the next IF.

## A Longer Chain

Try this one:

```
1. IF a number ends in 0, THEN it's even.
2. IF a number is even, THEN it's divisible by 2.
3. IF a number is divisible by 2, THEN you can split it into two equal parts.

Therefore: IF a number ends in 0, THEN you can split it into two equal parts.
```

Let's test it with 30:
- 30 ends in 0 (check!)
- So 30 is even (by rule 1)
- So 30 is divisible by 2 (by rule 2)
- So we can split 30 into two equal parts: 15 and 15 (by rule 3)

The chain worked!

## What If the IF Part Is False?

Important: If-then statements only tell you what happens when the IF part is TRUE.

```
IF it rains, THEN the ground gets wet.
```

What if it DOESN'T rain?

The statement doesn't tell us. Maybe someone used a hose and the ground is wet anyway. Maybe it's dry. The statement is silent about this case.

Example:

```
IF you study, THEN you'll pass the test.
```

What if you DON'T study? The statement doesn't say. Maybe you'll still pass (you got lucky), maybe you won't. The if-then rule only applies when you DO study.

## The Contrapositive: A Logic Trick

Here's something clever. Look at this statement:

```
IF it's a dog, THEN it's an animal.
```

This is true. Now look at this statement:

```
IF it's NOT an animal, THEN it's NOT a dog.
```

This is also true! And it's logically equivalent to the first statement.

Why? Because if something isn't even an animal, there's no way it can be a dog.

This is called the contrapositive, and it's always true when the original statement is true.

```
Original: IF A, THEN B
Contrapositive: IF NOT B, THEN NOT A
```

Both are true together, or false together.

## Practice With If-Then

Let's practice identifying the parts:

```
Statement: "If you water plants, they grow."
IF part: you water plants
THEN part: they grow
```

```
Statement: "If a shape has four equal sides, it's a square."
IF part: a shape has four equal sides
THEN part: it's a square
```

Actually, that second one isn't quite right - a square needs four equal sides AND four right angles. But you get the idea of identifying the parts.

## Answers to Exercises

1. IF you eat food, THEN you can run.
   (The chain connects: food → energy → running)

2. IF part: you add 5 + 3
   THEN part: you get 8

3. FALSE. The original statement doesn't tell us what happens when it's NOT raining. The ground could still be wet (from a sprinkler) or dry.

4. Many possible answers. Example:
   - IF you turn on the oven, THEN it gets hot.
   - IF it gets hot, THEN you can bake cookies.
   - Therefore: IF you turn on the oven, THEN you can bake cookies.

5. Contrapositive: "IF it doesn't have feathers, THEN it's not a bird."
   (Both the original and contrapositive are true together.)
