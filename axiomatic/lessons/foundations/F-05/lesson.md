# F-05: What Is Space?

Welcome! You're about to learn what space really means. Not outer space (though that's included!), but the idea of "where things are." By the end of this lesson, you'll understand dimensions and how we describe locations. You can do this.

## Space Is "Where Things Are"

Right now, you're somewhere. This book is somewhere. Your chair is somewhere. Everything exists in a location.

Space is the "everywhere" that contains all locations. It's the arena where everything happens.

Think about your room:
- Your bed is in one corner
- Your desk is against the wall
- Your lamp is on the desk

Each object has a position in space. Space is what lets things be in different places.

## One Dimension: A Line

The simplest kind of space is a line.

```
←──────────•────────────────•──────────────→
           A                B
```

On a line, you can only move in two directions: left or right (or forward and back).

**Examples of one-dimensional space:**

A number line:
```
←─────────────────────────────────────→
  0   1   2   3   4   5   6   7   8
```

You can be at position 3, or position 7, but you can only move left or right along the line.

A train track (ignoring the width):
```
Station A ─────────────────── Station B
```

The train can go forward or backward, but it can't leave the track.

**One dimension = one way to move (plus its opposite)**

## Two Dimensions: A Flat Surface

Now let's add another direction. Instead of just left-right, we also have forward-back.

```
        North (forward)
            ↑
            |
West ←──────+──────→ East
(left)      |      (right)
            |
            ↓
        South (back)
```

This is two-dimensional space. It's flat, like a piece of paper or a tabletop.

**To describe a position in 2D space, you need TWO numbers:**

```
The treasure is:
- 5 steps East (right)
- 3 steps North (forward)
Position: (5, 3)
```

**Examples of two-dimensional space:**

A chessboard:
```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
  ... (8 rows, 8 columns)

Position: "The knight is at E4" (column E, row 4)
```

A map:
- "The park is 2 blocks east and 3 blocks north"
- You need both numbers to find it

**Two dimensions = two ways to move (left-right AND forward-back)**

## Three Dimensions: Our World

Our world has one more direction: up-down.

```
          Up
           ↑
           |
           |
    North  |
      ↑    |
      |    |
West ←┼────┼──→ East
      |    |
      ↓    |
    South  |
           |
           ↓
         Down
```

This is three-dimensional space. Everything you can see and touch exists in 3D space.

**To describe a position in 3D space, you need THREE numbers:**

```
The bird is:
- 10 feet East
- 5 feet North
- 20 feet Up
Position: (10, 5, 20)
```

**Examples of three-dimensional space:**

A room:
- 8 feet wide (left-right)
- 10 feet deep (forward-back)
- 9 feet tall (up-down)

Flying a drone:
- "The drone is 50 feet north, 30 feet west, and 40 feet up"

Playing in a swimming pool:
- You can swim left-right, forward-back, AND dive up-down

**Three dimensions = three ways to move (left-right AND forward-back AND up-down)**

## Regions: Parts of Space

A region is a portion of space. It's like drawing a boundary and saying "this part."

**Examples of regions:**

1D region (a segment of a line):
```
←───────[=======]──────→
        Region
```

2D region (an area):
```
+─────────+
|         |
|  Park   |  ← This region is the park
|         |
+─────────+
```

3D region (a volume):
```
    +───────+
   /       /|
  +───────+ |
  |       | |  ← This region is "inside the box"
  |       | +
  |       |/
  +───────+
```

**Regions help us talk about locations:**
- "Inside the house" (a 3D region)
- "Within city limits" (a 2D region on a map)
- "Between mile markers 5 and 8" (a 1D region on a highway)

## Distance in Space

Distance measures how far apart things are.

**In 1D:**
```
←───•─────────•────→
    A         B

Distance from A to B: count the steps between them
```

**In 2D:**
```
  •B
  |
  |
  •A

Distance from A to B: You could measure straight-line distance (as the crow flies) or walking distance (along a grid)
```

**In 3D:**
```
         •B (higher and to the right)
        /
       /
      •A

Distance from A to B: The straight-line distance through 3D space
```

The shortest distance between two points is always a straight line.

## Coordinates: Naming Positions

To precisely say where something is, we use coordinates (numbers that specify the location).

**1D coordinates:** Just one number
```
Position on a number line: 7
(You're at the spot marked 7)
```

**2D coordinates:** Two numbers (x, y)
```
Position on a grid: (3, 5)
(3 steps right, 5 steps up from the origin)
```

**3D coordinates:** Three numbers (x, y, z)
```
Position in space: (2, 4, 6)
(2 right, 4 forward, 6 up from the origin)
```

The origin is the starting point, usually marked (0, 0) in 2D or (0, 0, 0) in 3D.

You didn't need any special talent. You just needed to think clearly about directions and dimensions. That's all spatial thinking ever is.
